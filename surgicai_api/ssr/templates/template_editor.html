Template Fields

Fields are placeholders that can be filled in when creating operative notes from templates. There are two types of fields:

Regular Fields (Yellow):
Custom fields you create for any purpose. Examples: Patient Name, Surgeon Name, Procedure
AI Fields (Blue):
Predefined fields that can be auto-filled by AI. Examples: Postoperative Diagnosis, Procedures

Creating Fields:

  1. Click the "Insert Field" dropdown in the toolbar and select "Field" or "AI Field", or
  2. Type / to open the slash menu
  3. Select "Field" for custom fields or "AI Field" for predefined options
  4. Enter a descriptive name for your field
  5. The field will appear highlighted in your template

Field Behavior:

  • Fields are saved in the format [field: name] or [aifield: name]
  • Use Backspace to delete entire fields
  • Arrow keys navigate around fields as single units
  • Fields maintain their formatting when templates are used
Tips & Best Practices

Template Design:

  • Use descriptive field names (e.g., "Patient Name" not "Name")
  • Place fields where information will be filled in
  • Use AI fields for content that can be auto-generated
  • Keep static text separate from dynamic fields

Field Naming:

  • Use clear, specific names
  • Avoid abbreviations when possible
  • Consider how others will understand the field purpose
Example Template Structure:
PATIENT: [field: Patient Name] DATE: [field: Surgery Date] SURGEON: [field: Primary Surgeon] PROCEDURE: [aifield: Procedures] DIAGNOSIS: [aifield: Postoperative Diagnosis]
